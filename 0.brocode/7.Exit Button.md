# 🎓 **JavaFX Lesson: Logout / Exit Button with Confirmation Dialog**

Logging out or exiting an application is a **core feature** in desktop applications.  
In JavaFX, we implement this using:

- A **Button** in the UI (Logout)
    
- A **Confirmation Alert**
    
- Closing or hiding the **Stage**
    
- `setOnCloseRequest()` to intercept the window’s close button
    

Your code demonstrates a perfect example of this.

---

# 🧩 **1. How Logout Works in JavaFX**

A logout button should do two things:

1. Ask the user for confirmation
    
2. If OK → close the window (or go to login screen)
    
3. If Cancel → do nothing
    

We use **Alert(AlertType.CONFIRMATION)** for this.

---

# 🛠️ **2. The Logout Button in Scene.fxml**

```xml
<Button fx:id="logoutButton"
        layoutX="274.0"
        layoutY="188.0"
        mnemonicParsing="false"
        onAction="#logout"
        text="Logout" />
```

### Key Points:

- `onAction="#logout"` → calls the `logout()` method in the controller.
    
- The button triggers the confirmation popup.
    

---

# 🧠 **3. The Controller: Logout Logic**

```java
public void logout(ActionEvent event) {
        
    Alert alert = new Alert(AlertType.CONFIRMATION);
    alert.setTitle("Logout");
    alert.setHeaderText("You're about to logout!");
    alert.setContentText("Do you want to save before exiting?: ");

    if(alert.showAndWait().get() == ButtonType.OK){
        stage = (Stage) scenePane.getScene().getWindow();
        System.out.println("You successfully logged out!");
        stage.close();
    }
}
```

### ✔ Explanation:

### **① Create Confirmation Dialog**

```java
Alert alert = new Alert(AlertType.CONFIRMATION);
```

### **② Set title, header, and message**

```java
alert.setTitle("Logout");
alert.setHeaderText("You're about to logout!");
alert.setContentText("Do you want to save before exiting?: ");
```

This creates a standard logout popup.

### **③ showAndWait(): stop code until the user responds**

```java
alert.showAndWait().get()
```

### **④ If OK → close the stage**

```java
stage = (Stage) scenePane.getScene().getWindow();
stage.close();
```

- We access the window via the `AnchorPane`
    
- Closing the stage closes the application
    

---

# 🧱 **4. Main Window Close Button (X) Handling**

In the Main class:

```java
stage.setOnCloseRequest(event -> {
    event.consume();
    logout(stage);
});
```

### ✔ What this does:

When the user clicks the **X button** on the window:

- `event.consume()` stops the default close behavior
    
- `logout(stage)` is called
    
- The same confirmation alert appears
    
- If user clicks OK → window closes
    
- If Cancel → application stays open
    

This ensures **both logout button and X button behave the same.**

---

# 🔍 **5. logout() Method in Main**

```java
public void logout(Stage stage){    

    Alert alert = new Alert(AlertType.CONFIRMATION);
    alert.setTitle("Logout");
    alert.setHeaderText("You're about to logout!");
    alert.setContentText("Do you want to save before exiting?");

    if (alert.showAndWait().get() == ButtonType.OK){
        System.out.println("You successfully logged out");
        stage.close();
    } 
}
```

Difference from controller version:

- This method is used when clicking the **window’s close button**
    
- Instead of switching scenes, it directly closes the stage
