# **JavaFX Lesson: Using DatePicker and Displaying the Selected Date**

## **Objective**

Learn how to use a `DatePicker` in JavaFX to allow users to select a date and display it in a `Label` in a custom format.

---

## **1. Prerequisites**

- Basic understanding of Java and JavaFX.
    
- JavaFX set up in your IDE.
    
- Familiarity with FXML and controllers.
    

---

## **2. Key Concepts**

- **DatePicker:** A UI component that allows users to select a date.
    
- **LocalDate:** Java object representing a date without time.
    
- **DateTimeFormatter:** Formats dates into a human-readable string.
    
- **FXML Bindings:** Link Java fields to UI elements using `@FXML`.
    

---

## **3. Controller Code Example**

```java
package application;

import java.time.LocalDate;
import java.time.format.DateTimeFormatter;
import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.DatePicker;
import javafx.scene.control.Label;

public class Controller {

    @FXML
    private DatePicker myDatePicker;  // Linked to FXML DatePicker

    @FXML
    private Label myLabel;  // Linked to FXML Label

    public void getDate(ActionEvent event) {
        LocalDate myDate = myDatePicker.getValue();  // Get selected date

        if (myDate != null) {
            // Format the date as "MMM-dd-yyyy" (e.g., Nov-22-2025)
            String myFormattedDate = myDate.format(DateTimeFormatter.ofPattern("MMM-dd-yyyy"));
            myLabel.setText(myFormattedDate);  // Display in Label
        } else {
            myLabel.setText("No date selected");  // Handle null case
        }
    }
}
```

---

## **4. FXML Example**

Make sure your FXML matches your controller:

```xml
<AnchorPane xmlns:fx="http://javafx.com/fxml/1" fx:controller="application.Controller">
    <children>
        <DatePicker fx:id="myDatePicker" layoutX="50" layoutY="50" onAction="#getDate"/>
        <Label fx:id="myLabel" layoutX="50" layoutY="100" text="Select a date"/>
    </children>
</AnchorPane>
```

---

## **5. How It Works**

1. **User Interaction:**  
    The user clicks the `DatePicker` and selects a date.
    
2. **Event Handling:**  
    The `onAction` event triggers the `getDate` method in the controller.
    
3. **Retrieving the Date:**  
    `myDatePicker.getValue()` returns the selected `LocalDate`.
    
4. **Formatting the Date:**  
    `DateTimeFormatter.ofPattern("MMM-dd-yyyy")` converts the `LocalDate` to a string like `Nov-22-2025`.
    
5. **Displaying the Date:**  
    The formatted date is set on `myLabel`.
    

---

## **6. Tips & Best Practices**

- Always **check for `null`** to prevent exceptions if no date is selected.
    
- Use **readable date formats** (e.g., `dd MMM yyyy`) for better UX.
    
- Specify **locale** in `DateTimeFormatter` to ensure consistent month names:
    
    ```java
    DateTimeFormatter.ofPattern("MMM-dd-yyyy", Locale.ENGLISH);
    ```
    
- Keep your **FXML IDs and controller fields** consistent.
    

---

## **7. Summary**

- `DatePicker` allows easy date selection in JavaFX.
    
- Use `LocalDate` to handle dates and `DateTimeFormatter` to display them.
    
- Always handle cases when no date is selected.
    
- Linking `onAction` in FXML triggers your method automatically.
