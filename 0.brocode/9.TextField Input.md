# 📘 JavaFX Lesson: TextField Input Validation

## 🎯 **Goal of This Lesson**

Learn how to:

- Accept user input from a `TextField`
    
- Validate the input (check if it’s a number and meets conditions)
    
- Display feedback in a `Label`
    
- Handle exceptions safely
    

---

# 🧠 **2. Controller.java**

```java
package application;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.Button;
import javafx.scene.control.Label;
import javafx.scene.control.TextField;

public class Controller {

    @FXML
    private Label myLabel;

    @FXML
    private TextField myTextField;

    @FXML
    private Button myButton;

    int age;

    // Method triggered by button click
    public void submit(ActionEvent event) {
        try {
            // Convert text to integer
            age = Integer.parseInt(myTextField.getText());

            // Validate age
            if(age >= 18) {
                myLabel.setText("You are now signed up!");
            }
            else {
                myLabel.setText("You must be 18+");
            }
        } catch (NumberFormatException e) {
            // If input is not a number
            myLabel.setText("Enter only numbers plz");
        } catch (Exception e) {
            // Catch all other errors
            myLabel.setText("Error");
        }
    }
}
```

# 🧩 **3. Key Concepts**

### 🔹 3.1 `TextField`

- User enters input here
    
- `myTextField.getText()` retrieves the input as a `String`
    

### 🔹 3.2 Parsing Numbers

```java
age = Integer.parseInt(myTextField.getText());
```

- Converts string to integer
    
- Throws `NumberFormatException` if input is not numeric
    

### 🔹 3.3 Exception Handling

- `try/catch` ensures the program **doesn’t crash**
    
- Provides feedback for invalid input
    

### 🔹 3.4 Updating Labels

- `myLabel.setText()` changes the displayed text dynamically
    

---

# 🔹 4. Button Action

The `onAction="#submit"` in FXML connects the **button click** to the `submit()` method in the controller.

# 🚀 **6. Extension Ideas**

- Add keyboard Enter key to trigger submission
    
- Validate more complex inputs (email, phone number)
    
- Add CSS to highlight invalid fields
    
- Limit age input to a max value (e.g., 120)

# 🏁 **Summary**

- `TextField` captures user input
    
- `Integer.parseInt()` converts to number
    
- `try/catch` prevents crashes and validates input
    
- `Label` displays feedback dynamically
