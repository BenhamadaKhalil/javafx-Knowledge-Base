# 📘 JavaFX Lesson: Moving a Shape with a Controller

## 🎯 **Goal of This Lesson**

Learn how to control a **Circle** in JavaFX using buttons — moving it **up, down, left, and right** — by updating its `centerX` and `centerY` values.

---

# 📦 **1. The Controller Code**

```java
package application;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.shape.Circle;

public class Controller {

    @FXML
    private Circle myCircle;
    private double x;
    private double y;

    @FXML
    public void initialize() {
        // Initialize starting position from the FXML circle
        x = myCircle.getCenterX();
        y = myCircle.getCenterY();
    }

    public void up(ActionEvent e) {
        myCircle.setCenterY(y -= 10);
    }

    public void down(ActionEvent e) {
        myCircle.setCenterY(y += 10);
    }

    public void left(ActionEvent e) {
        myCircle.setCenterX(x -= 10);
    }

    public void right(ActionEvent e) {
        myCircle.setCenterX(x += 10);
    }
}
```

---

# 🧠 **2. Key Concepts Explained**

## 🔹 2.1 `@FXML`

This annotation connects Java variables and methods to elements defined in the **FXML file**.

Example:

```java
@FXML private Circle myCircle;
```

This means `myCircle` refers to the `<Circle>` in your FXML layout.

---

## 🔹 2.2 The `initialize()` Method

JavaFX automatically calls this method **after loading the FXML**.

Here we use it to capture the **starting position** of the circle:

```java
x = myCircle.getCenterX();
y = myCircle.getCenterY();
```

Without this step, `x` and `y` default to **0**, causing incorrect or glitchy movement.

---

## 🔹 2.3 Moving the Circle

Movement is done by changing:

- `centerX` → left/right
    
- `centerY` → up/down
    

Example:

```java
myCircle.setCenterY(y -= 10);
```

This moves the circle **10 pixels up**.

---

# 🎮 **3. How Each Button Works**

|Button|Method Called|What Happens|
|---|---|---|
|Up|`up()`|Decrease Y → moves up|
|Down|`down()`|Increase Y → moves down|
|Left|`left()`|Decrease X → moves left|
|Right|`right()`|Increase X → moves right|

Each method updates:

- the internal coordinate (`x`, `y`)
    
- and the circle's visual position
    

---

# 🧩 **4. Common Issues & Fixes**

### ❌ Problem: The circle jumps or starts at the wrong place

**Cause:** You didn’t initialize `x` and `y`.

### ✅ Fix:

Use the `initialize()` method.
