P# 📘 JavaFX Lesson: CheckBox + Image Toggle

## 🎯 **Goal of This Lesson**

Learn how to:

- Use a **CheckBox** in JavaFX
    
- Detect if it’s selected or not
    
- Dynamically update a **Label** and **ImageView**
    

This example simulates a **lightbulb switch**.

---

# 🏗 **1. Scene.fxml**

```xml
<AnchorPane prefHeight="400.0" prefWidth="371.0"
            xmlns="http://javafx.com/javafx/15.0.1"
            xmlns:fx="http://javafx.com/fxml/1"
            fx:controller="application.Controller">

   <children>
      <!-- CheckBox acting as a switch -->
      <CheckBox fx:id="myCheckBox" layoutX="125.0" layoutY="308.0" text="ON/OFF" 
                mnemonicParsing="false" onAction="#change">
         <font><Font size="22.0"/></font>
      </CheckBox>

      <!-- Label showing ON/OFF status -->
      <Label fx:id="myLabel" layoutX="162.0" layoutY="181.0" text="OFF">
         <font><Font size="27.0"/></font>
      </Label>

      <!-- Lightbulb Image -->
      <ImageView fx:id="myImageView" layoutX="120.0" layoutY="14.0" fitHeight="150" fitWidth="132"
                 pickOnBounds="true" preserveRatio="true">
         <image>
            <Image url="@lightbulb2.png"/>
         </image>
      </ImageView>
   </children>
</AnchorPane>
```

---

# 🧠 **2. Controller.java**

```java
package application;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.CheckBox;
import javafx.scene.control.Label;
import javafx.scene.image.Image;
import javafx.scene.image.ImageView;

public class Controller {

    @FXML
    private CheckBox myCheckBox;

    @FXML
    private Label myLabel;

    @FXML
    private ImageView myImageView;

    // Images for ON/OFF states
    Image myImage1 = new Image(getClass().getResourceAsStream("lightbulb1.png")); // ON
    Image myImage2 = new Image(getClass().getResourceAsStream("lightbulb2.png")); // OFF

    // Method triggered when CheckBox is clicked
    public void change(ActionEvent event) {
        if(myCheckBox.isSelected()) {
            myLabel.setText("ON");
            myImageView.setImage(myImage1);
        } else {
            myLabel.setText("OFF");
            myImageView.setImage(myImage2);
        }
    }
}
```

---

# 🧩 **3. Key Concepts**

### 🔹 3.1 CheckBox

- `CheckBox` is a **toggle component** (checked or unchecked)
    
- `myCheckBox.isSelected()` returns **true** if checked
    

### 🔹 3.2 Updating Label

```java
myLabel.setText("ON"); // or "OFF"
```

- Dynamically reflects the state of the checkbox
    

### 🔹 3.3 Updating ImageView

- Use `myImageView.setImage(myImage)` to change the picture
    
- Preload both images to switch instantly
    

### 🔹 3.4 Event Handling

- The FXML `onAction="#change"` links **CheckBox click** to the `change()` method in the controller
    

---

# 🧪 **4. How It Works**

|CheckBox State|Label Text|Image Shown|
|---|---|---|
|Checked|ON|lightbulb1.png|
|Unchecked|OFF|lightbulb2.png|

# 🚀 **5. Extension Ideas**

- Animate the lightbulb (fade in/out) when toggled
    
- Add multiple CheckBoxes to control multiple images
    
- Use `ToggleGroup` for mutually exclusive options
    
- Change the color of the Label dynamically
