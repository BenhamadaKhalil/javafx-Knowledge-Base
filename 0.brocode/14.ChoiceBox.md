# 🍱 JavaFX Lesson: Using ChoiceBox to Display Selected Items

## 🎯 **Objective**

Learn how to use a **ChoiceBox** in JavaFX to let users pick an item from a list and display the selection in a label.

---

## 🧩 **Components Used**

|Component|Purpose|
|---|---|
|🧺 **ChoiceBox**|Dropdown list for choosing an item|
|🏷️ **Label**|Displays the selected food|
|🔄 **Initializable**|Allows populating ChoiceBox when app starts|
|🎯 **ActionEvent**|Captures user selection|

---

## 🧠 **How It Works (Simple Flow)**

```
App starts → initialize() runs → food list added to ChoiceBox
User selects item → getFood() triggered → Label updates 🏷️
```

---

## 📌 **Controller Code**

```java
package application;

import java.net.URL;
import java.util.ResourceBundle;
import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.fxml.Initializable;
import javafx.scene.control.ChoiceBox;
import javafx.scene.control.Label;

public class Controller implements Initializable {

    @FXML
    private Label myLabel;

    @FXML
    private ChoiceBox<String> myChoiceBox;

    // 🍣 The list of food items to choose from
    private String[] food = {"pizza", "sushi", "ramen"};

    @Override
    public void initialize(URL location, ResourceBundle resources) {
        // 📥 Add food list items to the ChoiceBox
        myChoiceBox.getItems().addAll(food);

        // 🎯 When user selects something, call getFood()
        myChoiceBox.setOnAction(this::getFood);
    }

    public void getFood(ActionEvent event) {
        // 🍕 Get selected food
        String myFood = myChoiceBox.getValue();

        // 🏷️ Display it in label
        myLabel.setText(myFood);
    }
}
```

## 📝 **Key Notes**

### ✔️ `initialize()`

Runs automatically when the FXML loads.  
Perfect for setting up lists & event handlers.

### ✔️ Setting the Action

```java
myChoiceBox.setOnAction(this::getFood);
```

This means:  
➡️ When user makes a selection → call `getFood()`.

### ✔️ Getting the Selected Item

```java
String myFood = myChoiceBox.getValue();
```

### ✔️ Updating UI

```java
myLabel.setText(myFood);
```

---

## 🎉 **Summary**

- A **ChoiceBox** is a simple dropdown component.
    
- Items are added during the `initialize()` phase.
    
- When the user selects an item, `getFood()` updates the UI.
    
- Super useful for settings, menus, user preferences, etc.
