## 1️⃣ Download Scene Builder

- Go to [Gluon Scene Builder](https://gluonhq.com/products/scene-builder/)
    
- Download the `.exe` for Windows (or the package for your OS)
    
- Install it somewhere easy to remember, e.g.:
    
    ```
    C:\Program Files\Scene Builder\SceneBuilder.exe
    ```
    

---

## 2️⃣ Configuring Scene Builder

### **Eclipse**

1. **Create an FXML file**
    
    - Right-click project → New → Other → JavaFX → JavaFX FXML Document
        
    - Name it, e.g., `Main.fxml`
        
2. **Link Scene Builder**
    
    - `Window → Preferences → JavaFX → Scene Builder executable`
        
    - Browse to `SceneBuilder.exe`
        
3. **Open FXML in Scene Builder**
    
    - Right-click `Main.fxml` → Open → Scene Builder
        
4. **Add UI elements** and save (`File → Save`)
    
5. **Load FXML in your code**
    

```java
Parent root = FXMLLoader.load(getClass().getResource("Main.fxml"));
Scene scene = new Scene(root);
stage.setScene(scene);
stage.show();
```

> ⚠ Make sure `Main.fxml` is in the **resources folder** or same package as your class.  
> If it’s in resources: `getResource("/Main.fxml")`

---

### **IntelliJ**

1. **Link Scene Builder**
    
    - `File → Settings → Languages & Frameworks → JavaFX → Scene Builder path`
        
2. **Open FXML**
    
    - Right-click your `.fxml` file → Open in Scene Builder
        
3. **Add UI elements** and save
    
4. **Load FXML in `start()` method**
    

```java
Parent root = FXMLLoader.load(getClass().getResource("/sample.fxml"));
Scene scene = new Scene(root);
stage.setScene(scene);
stage.show();
```

---

### ⚡ Pro Tips

- Make sure the FXML file name matches exactly, including capitalization.
    
- If you get a **NullPointerException**, it usually means the FXML file path is wrong.
    
- Keep your FXML files in `src/main/resources` for Maven projects.
    
- Always use `/` at the start if the FXML is in the resources folder:
    
    ```java
    getClass().getResource("/Main.fxml")
    ```
