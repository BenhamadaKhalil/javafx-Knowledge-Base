## **JavaFX Stages Overview**

### 1. **What is a Stage?**

- Top-level container for a JavaFX GUI (similar to `JFrame` in Swing).
    
- Represents a **window**.
    
- Can be customized with **title**, **size**, **icon**, **position**, and **full-screen mode**.
    

---

### 2. **Core Components**

1. **Stage**
    
    - The window itself.
        
    - Can be shown using `stage.show()`.
        
    - Can have multiple stages if needed (`new Stage()`).
        
2. **Scene**
    
    - Container for all graphical content.
        
    - Added to a stage: `stage.setScene(scene)`.
        
    - Can have a **background color**: `new Scene(root, width, height, Color.BLACK)`.
        
3. **Scene Graph**
    
    - Hierarchical tree of **nodes** (buttons, text fields, images, etc.).
        
    - Root node is required (e.g., `Group root = new Group();`).
        

---

### 3. **JavaFX Application Structure**

- Main class extends `Application`.
    
- Override `start(Stage primaryStage)` method.
    
- Launch the application using `launch(args)` from the `main` method.
    

```java
public class HelloFX extends Application {
    public static void main(String[] args) {
        launch(args);
    }

    @Override
    public void start(Stage stage) {
        Group root = new Group();
        Scene scene = new Scene(root, 400, 300, Color.BLACK);
        stage.setScene(scene);
        stage.setTitle("Stage Demo Program");
        stage.show();
    }
}
```

---

### 4. **Customizing the Stage**

- **Set title:** `stage.setTitle("My App");`
    
- **Set icon:**
    

```java
Image icon = new Image("icon.png");
stage.getIcons().add(icon);
```

- **Set size:** `stage.setWidth(420); stage.setHeight(420);`
    
- **Disable resizing:** `stage.setResizable(false);`
    
- **Set position on screen:** `stage.setX(50); stage.setY(50);`
    
- **Full-screen mode:**
    

```java
stage.setFullScreen(true);
stage.setFullScreenExitHint("Can't escape unless you press Q");
stage.setFullScreenExitKeyCombination(KeyCombination.valueOf("Q"));
```

---

### 5. **Tips**

- Scene resizes automatically with the stage.
    
- Root nodes act like layout managers (Group, VBox, HBox, etc.).
    
- Always call `stage.show()` at the end of your `start` method.

## 🔥 Final clean code (ready to use)

```java
package application;

import javafx.application.Application;
import javafx.stage.Stage;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.paint.Color;

public class Main extends Application {

    public static void main(String[] args) {
        launch(args);
    }

    @Override
    public void start(Stage stage) throws Exception {

        Group root = new Group();
        Scene scene = new Scene(root, Color.BLACK);

        stage.setScene(scene);
        stage.show();
    }
}

```
