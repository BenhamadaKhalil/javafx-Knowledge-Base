# ⭐ **Cleaner & modern JavaFX version (recommended)**

JavaFX allows writing listeners in a shorter lambda format.

### **Controller.java**

```java
@Override
public void initialize(URL location, ResourceBundle resources) {

    myListView.getItems().addAll(food);

    myListView.getSelectionModel().selectedItemProperty().addListener(
        (observable, oldValue, newValue) -> {

            currentFood = newValue;
            myLabel.setText(currentFood);
        }
    );
}
```

### ✔ Improvements

- Uses a lambda → shorter and more readable
    
- Directly uses `newValue` instead of calling `getSelectedItem()`
    
- Works exactly the same
