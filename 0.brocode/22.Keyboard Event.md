# 🎮 JavaFX Keyboard Event Handling – Full Lesson

This lesson explains how to **capture keyboard input in JavaFX** and use it to trigger movement actions inside a controller.  
We use the keys **W, A, S, D** to simulate movement.

---

## 📌 2. Main.java – Loading the Scene & Listening for Keyboard Input

```java
package application;

import java.io.IOException;
import javafx.application.Application;
import javafx.event.EventHandler;
import javafx.fxml.FXMLLoader;
import javafx.stage.Stage;
import javafx.scene.Parent;
import javafx.scene.Scene;
import javafx.scene.input.KeyEvent;

public class Main extends Application {

    @Override
    public void start(Stage stage) throws IOException {

        FXMLLoader loader = new FXMLLoader(getClass().getResource("Scene.fxml"));
        Parent root = loader.load();

        // Get the controller from the FXML
        Controller controller = loader.getController();

        Scene scene = new Scene(root);

        // Listen for keyboard input
        scene.setOnKeyPressed(new EventHandler<KeyEvent>() {

            @Override
            public void handle(KeyEvent event) {

                switch (event.getCode()) {

                    case W:
                        controller.moveUp();
                        break;
                    case S:
                        controller.moveDown();
                        break;
                    case A:
                        controller.moveLeft();
                        break;
                    case D:
                        controller.moveRight();
                        break;
                    default:
                        break;
                }
            }
        });

        stage.setScene(scene);
        stage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
```

### ✅ Key Concepts

- **FXMLLoader** loads the UI and gives access to the controller.
    
- **scene.setOnKeyPressed(...)** listens for key events.
    
- **event.getCode()** checks which key was pressed.
    
- **controller.moveUp()** calls functions inside the controller.

## 📌 3. Controller.java – Movement Methods

```java
package application;

public class Controller {

    public void moveUp() {
        System.out.println("MOVIN' UP!");
    }

    public void moveDown() {
        System.out.println("MOVIN' DOWN!");
    }

    public void moveLeft() {
        System.out.println("MOVIN' LEFT!");
    }

    public void moveRight() {
        System.out.println("MOVIN' RIGHT!");
    }
}
```

### ✅ This controller is simple:

- It contains **4 functions**, one for each direction.
    
- Each function prints a message to the console.
    
- In a real game, these would move a character or UI element.

## 📌 4. How It Works – Step by Step

### **Step 1:** Launch the app

JavaFX loads `Scene.fxml`.

### **Step 2:** Get the controller

We use:

```java
Controller controller = loader.getController();
```

### **Step 3:** Listen for keyboard input

JavaFX checks which key the user presses.

### **Step 4:** Use a switch statement

Depending on the key (W/A/S/D), call one of the controller methods.

### **Step 5:** Controller executes the action

Example output:

```
MOVIN' UP!
MOVIN' LEFT!
MOVIN' RIGHT!
```
