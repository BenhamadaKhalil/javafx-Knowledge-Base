_Based on your Controller.java code_

## 📌 Overview

JavaFX provides built-in animation classes that make it easy to animate UI elements such as images, buttons, and shapes.  
In this lesson, you will learn:

- What transitions are
    
- How to use `TranslateTransition`, `RotateTransition`, `FadeTransition`, and `ScaleTransition`
    
- How cycle counts, interpolators, and auto-reverse work
    
- How to structure animation code in a JavaFX controller
    

---

## 🧱 1. The Controller Structure

Every JavaFX `Controller` typically implements `Initializable` so code inside `initialize()` runs automatically when the FXML loads.

```java
public class Controller implements Initializable {

    @FXML
    private ImageView myImage;

    @Override
    public void initialize(URL arg0, ResourceBundle arg1) {
        // animations go here
    }
}
```

`myImage` is injected from the FXML file.

## 🚀 2. Translate Transition (Movement)

### ✔ What it does

Moves the node (your image) from one position to another.

### ✔ Code

```java
TranslateTransition translate = new TranslateTransition();
translate.setNode(myImage);
translate.setDuration(Duration.millis(1000));
translate.setCycleCount(TranslateTransition.INDEFINITE);
translate.setByX(500);
translate.setByY(-250);
translate.setAutoReverse(true);
translate.play();
```

### ✔ Explanation

- `setByX(500)` → move 500px to the right
    
- `setByY(-250)` → move 250px up
    
- `setCycleCount(INDEFINITE)` → infinite loop
    
- `setAutoReverse(true)` → goes back to the starting point
    
- `play()` → starts the animation

## 🔄 3. Rotate Transition (Rotation)

### ✔ What it does

Rotates a node around an axis (default is Z axis).

### ✔ Code

```java
RotateTransition rotate = new RotateTransition();
rotate.setNode(myImage);
rotate.setDuration(Duration.millis(500));
rotate.setCycleCount(RotateTransition.INDEFINITE);
rotate.setInterpolator(Interpolator.LINEAR);
rotate.setByAngle(360);
rotate.setAxis(Rotate.Z_AXIS);
rotate.play();
```

### ✔ Explanation

- Rotates 360 degrees every 0.5 seconds
    
- `Interpolator.LINEAR` → smooth constant rotation
    
- Infinite looping
    

---

## 🌫 4. Fade Transition (Opacity)

### ✔ What it does

Makes the image gradually disappear and reappear.

```java
FadeTransition fade = new FadeTransition();
fade.setNode(myImage);
fade.setDuration(Duration.millis(1000));
fade.setCycleCount(FadeTransition.INDEFINITE);
fade.setInterpolator(Interpolator.LINEAR);
fade.setFromValue(0);
fade.setToValue(1);
fade.play();
```

### ✔ Explanation

- `from 0 → to 1` → fades from invisible to fully visible
    
- Loop repeats forever
    

---

## 🔍 5. Scale Transition (Zoom In/Out)

### ✔ What it does

Changes the size of the node — zoom in / zoom out.

```java
ScaleTransition scale = new ScaleTransition();
scale.setNode(myImage);
scale.setDuration(Duration.millis(1000));
scale.setCycleCount(ScaleTransition.INDEFINITE);
scale.setInterpolator(Interpolator.LINEAR);
scale.setByX(2.0);
scale.setByY(2.0);
scale.setAutoReverse(true);
scale.play();
```

### ✔ Explanation

- `setByX(2.0)` → doubles width
    
- `setByY(2.0)` → doubles height
    
- `AutoReverse` → grows then shrinks

## 🧠 Summary Table

|Transition Type|Effect|Main Methods|
|---|---|---|
|`TranslateTransition`|Moves node|`setByX()`, `setByY()`, `setAutoReverse()`|
|`RotateTransition`|Rotates|`setByAngle()`, `setAxis()`|
|`FadeTransition`|Changes visibility|`setFromValue()`, `setToValue()`|
|`ScaleTransition`|Grows/shrinks|`setByX()`, `setByY()`|

---

## 🎯 Final Notes

- You can run multiple animations **at the same time** on the same node.
    
- You can combine animations using **ParallelTransition** or **SequentialTransition** to create complex effects.
    
- Using animations in the `initialize()` method ensures they start immediately when the scene loads.
