# 🌡️ JavaFX Lesson: Controlling a Slider & Updating a Label

## 🎯 **Objective**

Learn how to use a **Slider** in JavaFX and update a **Label** in real time as the slider moves — perfect for temperature controls, volume controls, brightness settings, etc.

---

# 🧩 Components Involved

|Component|Purpose|
|---|---|
|🎚️ **Slider**|The user drags it to change the value|
|🏷️ **Label**|Displays the current slider value (°C)|
|🔄 **ChangeListener**|Detects slider movement|
|🔧 **Initializable**|Sets initial UI values on startup|

---

# 📌 Controller Code (Explained)

```java
package application;

import java.net.URL;
import java.util.ResourceBundle;
import javafx.beans.value.ChangeListener;
import javafx.beans.value.ObservableValue;
import javafx.fxml.FXML;
import javafx.fxml.Initializable;
import javafx.scene.control.Label;
import javafx.scene.control.Slider;

public class Controller implements Initializable {

    @FXML
    private Label myLabel;

    @FXML
    private Slider mySlider;

    int myTemperature;

    @Override
    public void initialize(URL url, ResourceBundle resources) {

        // 🔥 1. Get initial slider value when app starts
        myTemperature = (int) mySlider.getValue();
        myLabel.setText(myTemperature + "°C");

        // 🎚️ 2. Add a listener to detect slider movement
        mySlider.valueProperty().addListener(new ChangeListener<Number>() {
            @Override
            public void changed(ObservableValue<? extends Number> observable,
                                Number oldNumber,
                                Number newNumber) {

                // 🌡️ 3. Update temperature when slider value changes
                myTemperature = (int) mySlider.getValue();
                myLabel.setText(myTemperature + "°C");
            }
        });
    }
}
```

---

# 🧠 How It Works (Step-By-Step)

## 1️⃣ **App starts → initialize() runs**

- JavaFX loads your FXML
    
- Injects the Slider and Label
    
- Calls `initialize()`
    

## 2️⃣ **Initial value setup**

```java
mySlider.getValue()
```

Gets the default slider value (set in FXML as `value="50.0"`).  
Then updates the label:

```
50°C
```

## 3️⃣ **Detect slider movement**

The listener watches the slider:

```java
mySlider.valueProperty().addListener(...)
```

Whenever the user drags the slider:

```
oldValue → newValue → listener reacts → label updates
```

## 4️⃣ **Label updates instantly**

```java
myLabel.setText(myTemperature + "°C");
```

This creates a **live temperature display**.

# 📝 Why Use a ChangeListener?

Because unlike a button or choice box, a slider **updates continuously** as the user moves it.  
A simple `onAction` would only trigger when the user _releases_ the slider — not when it's moving.

So the listener lets us update the UI **live**.

---

# 🧠 Quick Memory Trick

**“Slider moves → listener listens → label updates.”** 🎧➡️📝

# 🎉 Summary

- Use `initialize()` to set the first label value.
    
- Attach a `ChangeListener` to update UI as slider moves.
    
- Cast slider value to an `int` because sliders return `double`.
    
- Perfect for temperatures, sound levels, percentages, etc.
