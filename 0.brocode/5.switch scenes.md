# 🎓 **JavaFX Lesson: Multi-Scene Navigation Using FXML & Controllers**

In this lesson, we will learn how to:

1. Start a JavaFX application
    
2. Load an FXML file
    
3. Use controllers to switch scenes
    
4. Understand how JavaFX manages windows (Stages), layouts (Scenes), and UI files (FXML)
    

---

# 🧩 **1. The Main JavaFX Application**

```java
public class Main extends Application {

    @Override
    public void start(Stage stage) {
        try {
            Parent root = FXMLLoader.load(getClass().getResource("Scene1.fxml"));
            Scene scene = new Scene(root);
            stage.setScene(scene);
            stage.show();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    public static void main(String[] args) {
        launch(args);
    }
}
```

### ✔ What this class does:

### **1. Extends Application**

Every JavaFX app must extend `Application`.  
JavaFX will automatically call the `start()` method.

---

### **2. The start() Method**

JavaFX gives you a **Stage** (window) when the app starts.

Think of it like:

- **Stage** = the window
    
- **Scene** = the screen inside the window
    
- **Parent / FXML** = the UI elements on that screen
    

Inside `start()`:

### 🔸 Load the UI file (FXML)

```java
Parent root = FXMLLoader.load(getClass().getResource("Scene1.fxml"));
```

This loads `Scene1.fxml` and builds the UI.

---

### 🔸 Create a Scene

```java
Scene scene = new Scene(root);
```

A Scene represents everything you see inside a window.

---

### 🔸 Attach the Scene to the Stage

```java
stage.setScene(scene);
stage.show();
```

`show()` makes the window appear on the screen.

---

# 🧠 Summary of Main.java

The Main class ONLY does one job:

> **Start the app and show the first scene (`Scene1.fxml`)**

# 🎮 **2. The Scene Controller (Switching Between Scenes)**

```java
public class SceneController {

    private Stage stage;
    private Scene scene;
    private Parent root;

    public void switchToScene1(ActionEvent event) throws IOException {
        root = FXMLLoader.load(getClass().getResource("Scene1.fxml"));
        stage = (Stage)((Node)event.getSource()).getScene().getWindow();
        scene = new Scene(root);
        stage.setScene(scene);
        stage.show();
    }

    public void switchToScene2(ActionEvent event) throws IOException {
        root = FXMLLoader.load(getClass().getResource("Scene2.fxml"));
        stage = (Stage)((Node)event.getSource()).getScene().getWindow();
        scene = new Scene(root);
        stage.setScene(scene);
        stage.show();
    }
}
```

This class is responsible for switching between different scenes.

---

# 🧩 **How Scene Switching Works**

## ✔ Step 1: Load the new FXML file

```java
root = FXMLLoader.load(getClass().getResource("Scene1.fxml"));
```

This loads the UI layout for the scene you want to switch to.

---

## ✔ Step 2: Get the current Stage (the window)

```java
stage = (Stage)((Node)event.getSource()).getScene().getWindow();
```

Explanation:

- The button you clicked is the **source** of the event
    
- Get its scene → get its window → this gives you the Stage
    

This is how JavaFX lets us change the window’s content.

---

## ✔ Step 3: Create a new Scene

```java
scene = new Scene(root);
```

We build a new screen using the FXML file.

---

## ✔ Step 4: Replace the old scene

```java
stage.setScene(scene);
stage.show();
```

This updates the window to show the new layout.

---

# 🎯 Summary of SceneController

This class does one thing:

> **When a button is clicked, switch the window to a different FXML screen.**

---

# 🎨 **3. How This Works With Scene Builder**

Your FXML files (Scene1.fxml & Scene2.fxml):

- Contain the UI built in Scene Builder
    
- Each FXML uses:
    

```xml
fx:controller="application.SceneController"
```

So when you click a button in the UI, JavaFX knows to call:

```
switchToScene1()
switchToScene2()
```

depending on what you set in the button’s `onAction`.

# 🎓 Final Conceptual Summary

### ✔ JavaFX Architecture in your project:

|Part|What it Does|
|---|---|
|**Main.java**|Starts the app, shows the first scene|
|**FXML files**|Store the UI layout created in Scene Builder|
|**SceneController.java**|Controls navigation between scenes|
|**Stage**|The main application window|
|**Scene**|The content displayed inside the window|
|**Parent**|The root node of the UI loaded from FXML|
