# 📘 JavaFX Lesson: RadioButtons and ToggleGroup

## 🎯 **Goal of This Lesson**

Learn how to:

- Use **RadioButtons** in JavaFX
    
- Group them with **ToggleGroup** so only **one can be selected at a time**
    
- Dynamically update a **Label** based on selection
    

This example simulates **choosing your favorite food**.

---

# 🏗 **1. Scene.fxml**

```xml
<AnchorPane prefHeight="400.0" prefWidth="488.0"
            xmlns="http://javafx.com/javafx/15.0.1"
            xmlns:fx="http://javafx.com/fxml/1"
            fx:controller="application.Controller">

    <children>
        <!-- RadioButton 1 -->
        <RadioButton fx:id="rButton1" layoutX="14.0" layoutY="183.0"
                     text="pizza" mnemonicParsing="false" onAction="#getFood">
            <font><Font size="24.0"/></font>
            <toggleGroup>
                <ToggleGroup fx:id="food"/>
            </toggleGroup>
        </RadioButton>

        <!-- RadioButton 2 -->
        <RadioButton fx:id="rButton2" layoutX="196.0" layoutY="183.0"
                     text="sushi" mnemonicParsing="false" onAction="#getFood" toggleGroup="$food">
            <font><Font size="24.0"/></font>
        </RadioButton>

        <!-- RadioButton 3 -->
        <RadioButton fx:id="rButton3" layoutX="364.0" layoutY="183.0"
                     text="ramen" mnemonicParsing="false" onAction="#getFood" toggleGroup="$food">
            <font><Font size="24.0"/></font>
        </RadioButton>

        <!-- Label to show selection -->
        <Label fx:id="myLabel" alignment="CENTER" layoutX="-1.0" layoutY="53.0"
               prefHeight="17.0" prefWidth="488.0" text="Select what you want to eat">
            <font><Font size="36.0"/></font>
        </Label>
    </children>
</AnchorPane>
```

---

# 🧠 **2. Controller.java**

```java
package application;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.Label;
import javafx.scene.control.RadioButton;

public class Controller {

    @FXML
    private Label myLabel;

    @FXML
    private RadioButton rButton1, rButton2, rButton3;

    // Triggered when a RadioButton is clicked
    public void getFood(ActionEvent event) {
        if(rButton1.isSelected()) {
            myLabel.setText(rButton1.getText());
        }
        else if(rButton2.isSelected()) {
            myLabel.setText(rButton2.getText());
        }
        else if(rButton3.isSelected()) {
            myLabel.setText(rButton3.getText());
        }
    }
}
```

# 🧩 **3. Key Concepts**

### 🔹 3.1 RadioButton

- Represents a **single choice option**
    
- Can be **selected** or **deselected**
    

### 🔹 3.2 ToggleGroup

- Groups multiple RadioButtons
    
- Ensures **only one button can be selected at a time**
    
- In FXML:
    

```xml
<toggleGroup>
    <ToggleGroup fx:id="food"/>
</toggleGroup>
```

- Then reference the same group in other RadioButtons: `toggleGroup="$food"`
    

### 🔹 3.3 Getting the Selected RadioButton

- `rButton1.isSelected()` → returns **true** if this button is selected
    
- Use `getText()` to display its label in a `Label`

✔ RadioButtons allow **single choice selection**
