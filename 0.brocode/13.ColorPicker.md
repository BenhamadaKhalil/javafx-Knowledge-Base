# 🎨 **JavaFX Lesson: Using ColorPicker to Change Background Color**

## **Objective**

Learn how to use a `ColorPicker` in JavaFX to let users select a color and **dynamically change the background color of a Pane**.

---

## **1. Prerequisites**

- Basic knowledge of Java and JavaFX.
    
- Familiarity with FXML and controllers.
    
- Understanding of JavaFX layout panes (e.g., `Pane`).
    

---

## **2. Key Components**

|Component|Purpose|
|---|---|
|🎨 `ColorPicker`|Allows the user to choose a color from a palette.|
|🖼 `Pane`|Container whose background color will change.|
|🎯 `ActionEvent`|Triggered when the user selects a new color.|

---

## **3. Controller Code Example**

```java
package application;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.ColorPicker;
import javafx.scene.layout.Background;
import javafx.scene.layout.BackgroundFill;
import javafx.scene.layout.Pane;
import javafx.scene.paint.Color;

public class Controller {

    @FXML
    private Pane myPane;  // 🖼 The pane we will change the color of

    @FXML
    private ColorPicker myColorPicker;  // 🎨 Color picker component

    // 🔄 This method changes the background color of the pane
    public void changeColor(ActionEvent event) {
        Color myColor = myColorPicker.getValue(); // Get selected color
        myPane.setBackground(new Background(
            new BackgroundFill(myColor, null, null) // Apply color
        ));
    }
}
```

---

## **4. FXML Example**

```xml
<AnchorPane xmlns:fx="http://javafx.com/fxml/1" fx:controller="application.Controller">
    <children>
        <Pane fx:id="myPane" layoutX="50" layoutY="50" prefWidth="200" prefHeight="200" />
        <ColorPicker fx:id="myColorPicker" layoutX="50" layoutY="270" onAction="#changeColor"/>
    </children>
</AnchorPane>
```

---

## **5. How It Works Step-by-Step**

1. 🎨 **User picks a color** from the `ColorPicker`.
    
2. 🔔 **`onAction` triggers** the `changeColor` method.
    
3. 🌈 **Get the selected color:**
    
    ```java
    Color myColor = myColorPicker.getValue();
    ```
    
4. 🖼 **Update Pane background:**
    
    ```java
    myPane.setBackground(new Background(new BackgroundFill(myColor, null, null)));
    ```
    
5. ✅ **Result:** The pane instantly updates with the chosen color.
    

---

## **6. Tips & Tricks**

- Use `null` for `CornerRadii` and `Insets` if you want a **plain background**.
    
- You can add `CornerRadii` to make rounded corners:
    
    ```java
    new BackgroundFill(myColor, new CornerRadii(10), null)
    ```
    
- Great for creating **customizable UI themes** or **color picker apps**.
    

---

## **7. Visual Summary**

```
[ColorPicker 🎨]  ---> onAction ---> Controller.changeColor() ---> Pane 🖼 background updates
```

---

✅ **Summary:**

- `ColorPicker` lets users select colors.
    
- The `Pane` background can be dynamically updated with the selected color.
    
- Using `BackgroundFill` makes it easy to apply colors programmatically.
