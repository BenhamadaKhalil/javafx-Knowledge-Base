# 📦 1. What Is a TreeView?

A **TreeView** is a collapsible list that displays hierarchical data:

```
Files
 ├── Pictures
 │     ├── picture1
 │     └── picture2
 ├── Videos
 │     ├── video1
 │     └── video2
 └── Music
       ├── music1
       └── music2
```

Perfect for file explorers, menus, categories, etc.

# 🧠 2. Controller Structure

```java
public class Controller implements Initializable {
```

The controller:

- Connects Java code ↔ FXML UI
    
- Runs `initialize()` before the UI displays
    
- Uses `@FXML` to link UI elements
    

```java
@FXML
private TreeView<String> treeView;
```

# 🌱 3. Creating Tree Items

### ➤ Root Node

```java
TreeItem<String> rootItem = new TreeItem<>("Files");
```

This is the top-level node.

### ➤ Branches (Folders)

```java
TreeItem<String> branchItem1 = new TreeItem<>("Pictures");
TreeItem<String> branchItem2 = new TreeItem<>("Videos");
TreeItem<String> branchItem3 = new TreeItem<>("Music");
```

### ➤ Leaves (Files)

```java
TreeItem<String> leafItem1 = new TreeItem<>("picture1");
...
TreeItem<String> leafItem6 = new TreeItem<>("music2");
```

### ➤ Add leaves to branches

```java
branchItem1.getChildren().addAll(leafItem1, leafItem2);
branchItem2.getChildren().addAll(leafItem3, leafItem4);
branchItem3.getChildren().addAll(leafItem5, leafItem6);
```

### ➤ Add branches to root

```java
rootItem.getChildren().addAll(branchItem1, branchItem2, branchItem3);
```

### ➤ Set the root into the TreeView

```java
treeView.setRoot(rootItem);
```

# 🖱️ 5. Detecting Selected Items

FXML:

```xml
onMouseClicked="#selectItem"
```

Controller:

```java
public void selectItem() {
    TreeItem<String> item = treeView.getSelectionModel().getSelectedItem();
    if(item != null) {
        System.out.println(item.getValue());
    }
}
```

### 🧠 How it works:

1. User clicks on a node
    
2. The method gets the **selected TreeItem**
    
3. Prints the value (ex: `"picture1"`)
    

💡 You can replace `System.out.println()` with:

- Changing a label
    
- Loading a file
    
- Displaying an image
    
- Opening a new window
    

---

# 📄 6. FXML Structure

```xml
<TreeView fx:id="treeView"
          onMouseClicked="#selectItem"
          prefHeight="347.0"
          prefWidth="343.0" />
```

Important:

- `fx:id="treeView"` links to the controller `treeView`
    
- `onMouseClicked="#selectItem"` calls the method when clicked
