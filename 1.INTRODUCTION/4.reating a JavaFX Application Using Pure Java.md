# 🎬 Lesson 2 – Creating a JavaFX Application Using Pure Java

## 🧠 Overview

In this lesson, we’ll recreate the same “Hello Application” you built using **FXML**, but **entirely with Java code** — no FXML files.  
This helps you understand how the **JavaFX public API** works and how to structure your UI manually in Java.

---

## 🧹 Step 1 – Clean Up the Project

Before starting:

- Delete the **FXML** file (`hello-view.fxml`).
    
- Delete the **HelloController.java** file.
    
- Keep only the **HelloApplication.java** (main file).
    
- Don’t delete the **package**.
    
- If you rename files, make sure you update all references (in code and `pom.xml`).
    

---

## 🏗️ Step 2 – Create the Main Application Class

1. Start with a normal Java class:
    
    ```java
    public class HelloApplication extends Application {
        @Override
        public void start(Stage stage) {
            // code here
        }
    
        public static void main(String[] args) {
            launch(args);
        }
    }
    ```
    
2. The `Application` class is the **entry point** into the JavaFX world.  
    It requires you to **override the `start()` method**, which defines what happens when your app launches.
    

---

## 🎭 Step 3 – The Theatre Metaphor (JavaFX Structure)

Think of a JavaFX program like a **theatre**:

|Concept|Description|
|---|---|
|**Stage**|The window of your app.|
|**Scene**|The content inside the window.|
|**Controls**|The actors (buttons, labels, text fields, etc.).|
|**Layout**|The stage setup that arranges the controls (like VBox, HBox, GridPane…).|

Flow:

1. Create a **Stage**
    
2. Create a **Scene**
    
3. Add **controls** (buttons, labels…)
    
4. Add the **Scene** to the **Stage**
    
5. Show the **Stage**
    

---

## 🧩 Step 4 – Build the Scene Programmatically

1. Create a **root layout** (like VBox):
    
    ```java
    VBox root = new VBox();
    ```
    
2. Create your **controls**:
    
    ```java
    Label text = new Label();
    Button button = new Button("Click Me");
    ```
    
3. Add the controls to the layout:
    
    ```java
    root.getChildren().addAll(text, button);
    ```
    
    - `getChildren()` returns an **ObservableList** of nodes.
        
    - `addAll()` adds multiple elements at once.
        
    - “Observable” means you can listen for changes (like when new controls are added).
        

---

## 🖼️ Step 5 – Create and Set the Scene

```java
Scene scene = new Scene(root, 400, 300);
stage.setScene(scene);
stage.show();
```

Now the stage (window) is ready and displays your layout.

---

## 🎯 Step 6 – Center and Style the Layout

1. Align components to the center:
    
    ```java
    root.setAlignment(Pos.CENTER);
    ```
    
2. Add padding:
    
    ```java
    root.setPadding(new Insets(20, 20, 20, 20));
    ```
    
3. Add spacing between elements:
    
    ```java
    root.setSpacing(20);
    ```
    
4. Give the stage a title:
    
    ```java
    stage.setTitle("JavaFX Application");
    ```
    

---

## ⚡ Step 7 – Add Button Functionality

Use a **lambda expression** to define what happens when the button is clicked:

```java
button.setOnAction(e -> text.setText("Welcome to JavaFX Application!"));
```

When you run the program, the label updates when you click the button.

---

## ✅ Step 8 – Full Example

```java
package com.example.helloworld;

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.control.Label;
import javafx.scene.layout.VBox;
import javafx.geometry.Insets;
import javafx.geometry.Pos;
import javafx.stage.Stage;

public class HelloApplication extends Application {

    @Override
    public void start(Stage stage) {
        Label text = new Label();
        Button button = new Button("Click Me");

        button.setOnAction(e -> text.setText("Welcome to JavaFX Application!"));

        VBox root = new VBox();
        root.setAlignment(Pos.CENTER);
        root.setPadding(new Insets(20));
        root.setSpacing(20);
        root.getChildren().addAll(text, button);

        Scene scene = new Scene(root, 400, 300);
        stage.setScene(scene);
        stage.setTitle("JavaFX Application");
        stage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
```

---

## 💬 Summary

- You learned how to **build a JavaFX app without FXML**.
    
- You created everything directly in Java — layout, scene, and event handling.
    
- While this method works, it mixes **UI design** and **logic**, making it harder to maintain.
    
- That’s why **FXML** and **controllers** are preferred for larger projects — they follow the **MVC pattern (Model–View–Controller)** and separate concerns.
